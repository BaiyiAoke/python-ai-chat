# python-ai-chat

这是一个用于调用AI API进行聊天功能的Python项目。

## 项目结构

```
python-ai-chat
├── src
│   ├── main.py
│   └── services
│       └── chat_service.py
├── .env
├── requirements.txt
└── README.md
```

## 安装依赖

在项目根目录下运行以下命令以安装所需的依赖项：

```
pip install -r requirements.txt
```

## 使用说明

1. 在`.env`文件中配置您的API密钥。
2. 运行`src/main.py`以启动聊天功能。
3. 按照提示输入您的消息，与AI进行交互。
